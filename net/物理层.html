# 五. 数据通信基本概念

  1. 消息 (message)：通信的目的是传递消息
  2. 数据 (data)：数据是运送消息的实体
  3. 信号 (signal)：信号是数据电气或电磁的表现
  4. 信道 (channel)：信道是传送信号的媒体
  1. 码元：在有时域的波形表示数字信号时，码元是一个离散值/波形/脉冲
  2. k进制码元：数字信号有k种离散值/波形/脉冲
  3. 码元传输速率：单位时间传输的码元数，单位是波特
  4. 信息传输速率：k * 码元传输速率

# 六. 数据/信号/信道的分类
