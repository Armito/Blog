# 一. Text 组件

继承自 StatelessWidget

# 1.1 style 参数

参数 值 参数 值 选项 备注
style TextStyle() 文字样式 fontSize double / 文字大小
style TextStyle() fontWeight FontWeight .w100~900, .bold... 文字粗细
style TextStyle() fontStyle FontStyle .normal, .italic 文字样式
style TextStyle() color Color / Colors .fromRGBO() / .write70... 文字颜色
style TextStyle() letterSpacing double / 字母间隔
style TextStyle() wordSpacing double / 单词间隔
style TextStyle() decoration TextDecoration .combine([]), .underline... 装饰线位置
style TextStyle() decorationColor Color / Colors .fromRGBO() / .write70... 装饰线颜色
style TextStyle() decorationStyle TextDecorationStyle .dashed, .wavy... 装饰线样式

# 1.2 其他参数

参数 值 参数 值 选项 备注
textAlign TextAlign .center, .right... / / 文本对齐
textDirection TextDirection .ltr, .rtl... / / 文本方向
textScaleFactor double / / / 文字缩放
overflow TextOverflow .ellipsis, .clip... / / 文字溢出
maxLines int / / / 最大行数

decoration BoxDecoration() border Border .all(), .only color, width, style 容器边框

# 三. Image 组件

Image.network() / image.asset()

参数 值 参数 值 选项 / 参数 值 / 参数 备注
fit BoxFit .contain, .cover, .fill... / / / 图片显示
repeat ImageRepeat .noRepeat, .repeatX... / / / 图片重复
alignment Alignment (double), .center, .bottomRight... / / / 图片对齐
color Color / Colors .fromRGBO() / .write70... / / / 图片容器颜色
colorBlendMode BlendMode .luminosity, .colorburn... / / / 图片与容器颜色混合
width double / / / / 图片宽度
height double / / / / 图片高度

注意: Image 的 width 和 height 只在没有宽高参数的组件中有效, 比如 ClipOval 组件 image.asset() 引入本地图片时, 需要在 pubspec.yaml 文件中配置: assets: images/xxx.jgp

# 四. ListView 组件

# 4.1 参数

参数 值 参数 值 选项 / 参数 值 / 参数 备注
padding EdgeInsets .all(), .fromLTRB(l, t, r, b), .only(l/t/r/b) double / / 列表内边距
children <Widget>[] / / / / 列表子组件
scrollDirection Axis .horizontal, .vertical / / / 列表排列方向
