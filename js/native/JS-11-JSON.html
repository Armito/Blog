# JSON

# 一. 语法

# 1. 安全的 JSON 值 (JSON-safe)

简单值: String, Number, Boolean, Null 对象: 对象的属性必须加双引号 数组

# 2. 不安全的 JSON 值

undefined, function, symbol 和包含循环引用 (对象之间相互引用，形成一个无限循环) 的对象 JSON.stringify() 在对象中遇到上述类型时会自动将其忽略，在数组中则会返回 null JSON.stringify() 遇到循环引用时会报错

# 二. 序列化

JSON.stringify(object, replacer, space)
  • replacer

如果是一个数组: 那么结果中将只包含数组中列出的属性

var book = { 
    "title": "Professional JavaScript", 
    "authors": [ 
    "Nicholas C. Zakas" 
    ], 
    edition: 3, 
    year: 2011 
    }; 
var jsonText = JSON.stringify(book, ["title", "edition"]);
// 结果
{"title": "Professional JavaScript", "edition": 3}

如果是一个函数: 函数接收两个参数, 属性(键)名和属性值. 根据属性(键)名可以知道应该如何处理要序列化的对象中的属性

var jsonText = JSON.stringify(book, function (key, value) { 
    switch(key) { 
        case "authors": 
            return value.join(",") 
        case "year": 
            return 5000; 
        case "edition": 
            return undefined; 
        default: 
            return value; 
    } 
});
// 结果
{"title": "Professional JavaScript", "authors": "Nicholas C. Zakas", "year": 5000}
  • space

如果是一个数值: 表示的是每个级别缩进的空格数. 最大缩进空格数为 10, 所有大于 10 的值都会自动转换为 10 如果是一个字符串: 则这个字符串将在 JSON 字符串中被用作缩进字符而不再使用空格

  • toJSON()

可以给任何对象添加 toJSON()方法, 从而得到自己所希望的 JSON 数据来序列化

  • 对象序列化的顺序

对象 toJSON() 方法的返回值/返回对象本身 => 返回值传入 replacer => 序列化 => 根据 space 格式化

# 三. 解析

JOSN.parse(sObject, reviver)
  • reviver

是一个函数, 函数接收两个参数, 属性(键)名和属性值. 根据属性(键)名可以知道应该如何处理要序列化的对象中的属性

var book = { 
    "title": "Professional JavaScript", 
    "authors": [ 
    "Nicholas C. Zakas" 
    ], 
    edition: 3, 
    year: 2011, 
    releaseDate: new Date(2011, 11, 1) 
}; 
var jsonText = JSON.stringify(book); 
var bookCopy = JSON.parse(jsonText, function(key, value){ 
    if (key == "releaseDate"){ 
    return new Date(value); 
    } else { 
        return value; 
    }
}); 
alert(bookCopy.releaseDate.getFullYear());
