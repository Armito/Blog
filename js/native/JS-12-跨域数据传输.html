# 跨域数据传输

  • 跨域原因

Ajax受限于浏览器的同源政策。

  • 同源政策 Same-origin Policy

请求协议protocol，域名host，端口port有一项不与自身服务器相同，即为非同源。

# 1. jsonp

JSON with Padding

  • 原理
  1. script 标签 src 属性的 get 请求不受同源政策限制；
  2. 服务器端返回一个函数调用的字符串，将数据以实参的方式传递；
  3. 客户端事先声明一个同名的全局函数，用形参接收数据，在函数体中处理数据
// 服务器端响应方式
res.send('fn({key: value})');
// 客户端请求与接收方式
<script>
    function fn(data) {
        console.log(data);
    }
</script>
<script src="http://..."></script>
  • 代码封装
function jsonp(options) {
    // 1. 生成唯一函数名
    let uniq = 'myJsonp' + Math.random().toString().replace('.', '');
    // 2. 将参数中的回调函数变成全局函数，并'命名'
    window[uniq] = options.success;
    // 3. 请求参数处理
    let params = '';
    if (options.data) {
        for (let k in options.data) {
            params += `&${k}=${options.data[k]}`;
        }
    }
    // 4. 请求地址拼接
    let path = `${options.url}?callback=${uniq}${params}`;
    // 5. 生成 script 标签
    let script = document.createElement('script');
    // 6. 添加请求地址到 src 属性
    script.src = path;
    // 7. 发送请求
    document.body.appendChild(script);
    // 8. 请求完毕后删除 script 标签
    script.onload = function () {
        this.remove();
    }
}

# 2. CORS

Cross-Origin Resource Sharing 跨域资源共享

  • 原理

客户端发送 Ajax 请求时，携带请求头 Origin 服务器端响应时，设置响应头 Access-Control-Allow-Origin

  • 限制
  1. 不能使用 setRequestHeader() 设置自定义请求头
  2. 不能发送和接收 cookie
  3. 调用 getAllResponseHeaders() 总返回空字符串
  • Preflighted 请求

升级版的 CORS, 这种请求使用 OPTIONS 方法, 发送下列头部

  1. Origin
  2. Access-Origin-Request-Method: 请求自身使用的方法
  3. Access-Origin-Request-Headers: 自定义的头部信息，多个头部以逗号分隔

服务器响应发送下列头部与浏览器进行沟通

  1. Access-Control-Allow-Origin:
  2. Access-Control-Allow-Methods: 允许的方法，多个方法以逗号分隔
  3. Access-Control-Allow-Headers: 允许的头部，多个头部以逗号分隔
  4. Access-Control-Max-Age: 应该将这个 Preflight 请求缓存多长时间(s)
  • 带 cookie 的请求

设置客户端请求是否携带 cookie: xhr.withCredentials = boolean; 设置服务器是否接收 cookie 信息: Access-Control-Allow-Credentials

# 3. 服务器代理

  • 原理

不同服务器间请求和响应数据不受同源限制； 客户端先向自身服务器发送 Ajax 请求，自身服务器再向其他服务器发送请求

# 4. WebSocket

先使用 http 建立连接, 再采用自己的协议, 一般用在网页聊天的场景下

// 创建连接
let socket = new WebSocket('wss://www.baidu.com')
socket.onopen = function () {
    console.log('连接成功');
}
socket.onerror = function () {
    console.log('连接失败');
}
socket.onclose = function () {
    console.log('连接关闭')
}
// 发送数据
socket.send(JSON.stringify(message));
// 接收数据
socket.onmessage = function (e) {
    let data = e.data;
}
