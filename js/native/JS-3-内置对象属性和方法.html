# 内置对象属性和方法

# 一. Object

# 1.1 Descriptor: 对象的属性特性

# 1.1.1 两种属性的属性特性

  • 对于数据属性
  1. value: 值 => 任意类型
  2. writable: 可写性 => Boolean
  3. enumerable: 可枚举性 => Boolean
  4. configurable: 可配置性 => Boolean
  • 对于存储器属性
  1. get: 读取 => Function
  2. set: 监视变化 => Function
  3. enumerable: 可枚举性 => Boolean
  4. configurable: 可配置性 => Boolean

# 1.1.2 属性特性的默认值

  • 对于点运算符添加的对象属性
  1. value: undefined
  2. writable: true
  3. enumerable: true
  4. configurable: true
  • 对于内置方法添加的对象属性
  1. value: undefined
  2. writable: false
  3. enumerable: false
  4. configurable: false

# 1.1.3 内置对象的属性特性

  1. value: 任意类型
  2. writable: true / fasle
  3. enumerable: false
  4. configurable: prototype 属性为 false, 其他均为 true

# 1.2 Object 的静态属性/方法

# 1.2.1 增删查改

# Object.create()

作用: 创建一个对象 参数: (要创建的对象的原型对象[, {要创建的对象的属性名 1: {属性特性 1}, 要创建的对象的属性名 2: {属性特性 2}...}]) 返回: 创建后的对象

let a = {x: 0};
let b = Object.create(a, {
  'name': {
    value: 'Pikachu', 
    writable: true, 
    enumable: false, 
    configable: true
  }, 
  'age': {value: 18}
});
b; // {name: "Pikachu", age: 18}
b.x; // 0
b.__proto__ == a; // true
# Object.setPrototypeOf()

作用: 设置目标对象的原型对象 参数: (目标对象[, 要设置的原型对象]) 返回: 设置后的目标对象

let a = {};
let b = {};
let c = Object.create(a);
c.__proto__ == a; // true
let d = Object.setPrototypeOf(c, b);
c.__proto__ == a; // false
c.__proto__ == b; // true
c == d; // true
# Object.getPrototypeOf()

作用: 获取目标对象的原型对象 参数: (目标对象) 返回: 目标对象的原型对象

let a = {};
let b = Object.create(a);
let c =  Object.getPrototypeOf(b);
a == c; // true
# Object.defineProperty()

作用: 1. 给目标对象添加属性, 并设置属性特性; 2. 给目标对象的目标属性设置特性 参数: (目标对象, 目标属性, {属性特性}) 返回: 设置后的目标对象

let a = {};
let b = Object.defineProperty(a, 'name', {
  value: 'Pikachu'
});
a.name; // 'Pikachu'
a == b; // true
# Object.defineProperties()

作用: 同上, 可以批量操作属性 参数: (目标对象, {目标属性 1: {属性特性 1}, 目标属性 2: {属性特性 2}...}) 返回: 设置后的目标对象

let a = {};
let b = Object.defineProperties(a, {
  'name': {
    value: 'Pikachu',
    enumerable: true
  },
  'age': {
    value: 18,
    enumerable: true
  }
});
Object.entries(a); // [["name", "Pikachu"], ["age", 18]]
a == b; // true

Object.defineProperty() 和 Object.defineProperties() 的属性特性修改规则:

  1. value: 由 writable 控制: true, 可以修改; false, 不能修改
  2. writable: 由 configurable 控制: ture, 可以修改; false, 能改为 false, 不能改为 true
  3. enumerable: 由 configurable 控制: true, 可以修改; false, 不能修改
  4. configurable: 由 configurable 控制: true, 可以修改; false, 不能修改
# Object.assign()

作用: 将多个对象自身的可枚举属性拷贝到目标对象 参数: (目标对象, 被拷贝属性的对象 1, 被拷贝属性的对象 2...) 返回: 处理后的目标对象

let a = {name: 'denden', age: 18};
let b = {name: 'Pikachu', hobby: 'ball'};
let c = Object.assign(b, a);
c == b; // true
b; // {name: "denden", hobby: "ball", age: 18}

# Object.values()

作用: 获取可枚举的属性值 参数: (目标对象) 返回: 一个若干值构成的数组

# Object.fromEntries()

作用: Object.entries() 的逆操作 参数: (可迭代的键值对列表) 返回: 一个对象

obj; // { foo: "bar", baz: 42 }
# Object.getOwnPropertyNames()

作用: 获取目标对象自身的字符串类型的键 参数: (目标对象) 返回: 一个若干键构成的数组

# Object.getOwnPropertySymbols()

作用: 获取目标对象自身的符号类型的键 参数: (目标对象) 返回: 一个若干键组成的数组

# Object.getOwnPropertyDescriptor()

作用: 获取目标对象自身的目标属性的特性 参数: (目标对象, 目标属性) 返回: 一个属性特性的对象

# Object.getOwnPropertyDescriptors()

作用: 获取目标对象自身的所有属性的特性 参数: (目标对象) 返回: 一个若干属性特性对象构成的对象

let a = {
  name: 'Pikachu',
  age: 18
};
Object.getOwnPropertyDescriptors(a); 
/*  
  { 
    name: {
      value: "Pikachu", writable: true, enumerable: true, configurable: true
    }, 
    age: {
      value: 18, writable: true, enumerable: true, configurable: true
    }
  } 
*/

# 1.2.3 属性限制

# Object.preventExtensions()

作用: 禁止给目标对象添加新属性 参数: (目标对象) 返回: 处理后的目标对象

let a = {
  name: 'Pikachu',
  age: 18
};
let b = Object.preventExtensions(a);
a.hobby = 'ball';
a.hobby; // undefined
a == b; // true
# Object.isExtensible()

作用: 判断目标对象是否可扩展属性 参数: (目标对象) 返回: 一个布尔值

# Object.seal()

作用: 禁止给目标对象添加新属性, 且所有自身属性的 configurable: false 参数: (目标对象) 返回: 处理后的目标对象

# Object.isSealed()

作用: 判断目标对象的属性是否被密封 参数: (目标对象) 返回: 一个布尔值

# Object.freeze()

作用: 禁止给目标对象添加新属性, 且所有自身属性的 writable: false, configurable: false 参数: (目标对象) 返回: 处理后的目标对象

# Object.isFrozen()

作用: 判断目标对象的属性是否被冻结 参数: (目标对象) 返回: 一个布尔值

  • 注意:
  1. 内置对象和自定义对象默认都是可扩展的
  2. 将对象设置为不可扩展/密封/冻结都是不可逆操作
  3. 这些方法只会让对象自身不可扩展, 其原型对象上扩展属性仍然可以继承
let a = {};
let b = Object.create(a);
b.name = 'denden';
Object.preventExtensions(b);
a.age = 20;
b.age; // 20

# 1.2.4 其他

# Object.is()

作用: 比较两个值是否严格相等 参数: (目标值 1, 目标值 2) 返回: 一个布尔值

// 基本等同于 === 比较, 特殊的:
NaN === NaN; // false
Object.is(NaN, NaN); // true
0 === -0; // true
Object.is(0, -0); // false

# 二. Function

# Function 的实例属性/方法

# Function.prototype.toString()

作用: 将函数转换为字符串 参数: 无 返回: 一个字符串

# Function.prototype.call()

作用: 改变函数 this 指向并调用函数 参数: (this 指向[, arg1, arg2...]) 返回: 被调用的函数的返回值

# Function.prototype.apply()

作用: 改变函数 this 指向并调用函数 参数: (this 指向[, [arg1, arg2...]]) 返回: 被调用的函数的返回值

# Function.prototype.bind()

作用: 改变函数 this 指向但不调用函数 参数: (this 指向[, arg1, arg2...]) 返回: 改变后的函数

// 封装 call(), apply(), bind()
Function.prototype.myCall = function (_this = window, ...val) {
    if (typeof _this != 'object') {
        _this = Object(_this);  
    } 
    _this.f = this;
    let result = _this.f(...val);
    delete _this.f;
    return result;
};
Function.prototype.myApply = function (_this = window, val) {
    if (typeof _this != 'object') {
        _this = Object(_this);
    }
    _this.f = this;
    let result = val ? _this.f(...val) : _this.f();
    delete _this.f;
    return result;
};
Function.prototype.myBind = function (_this = window, ...val) {
    if (typeof _this != 'object') {
        _this = Object(_this);
    }
    _this.f = this;
    return function () {
        let result = _this.f(...val);
        delete _this.f;
        return result;
    }
};

# 三. Array

# 3.1 Array 的静态属性和方法

# Array.isArray()

作用: 判断目标是否为数组 参数: (目标) 返回: 一个布尔值

# Array.from()

作用: 将伪数组对象或可迭代对象转变为真数组 参数: (类数组[, 处理新数组的回调函数[, 回调函数的 this 指向]]) 返回: 转变后的数组

Array.from([1, 2, 3], item => item * 2); // [2, 4, 6]

作用: 创建一个数组 参数: (元素 1, 元素 2...) 返回: 创建后的数组

Array.of(7); // [7]

# 3.2 Array 的实例属性和方法

# 3.2.1 增删改-原数组有变化

# Array.prototype.unshift()

作用: 添加一个或多个元素到数组的开头 参数: (元素 1, 元素 2...) 返回: 处理后的数组的长度

# Array.prototype.push()

作用: 添加一个或多个元素到数组的末尾 参数: (元素 1, 元素 2...) 返回: 处理后的数组的长度

# Array.prototype.shift()

作用: 删除数组中第一个元素 参数: 无 返回: 成功, 返回被删除的元素; 失败, 返回 undefined

# Array.prototype.pop()

作用: 删除数组中最后一个元素 参数: 无 返回: 成功, 返回被删除的元素; 失败, 返回 undefined

# Array.prototype.splice()

作用: 1. 添加元素; 2. 删除元素; 3. 替换元素 参数: (开始处索引[, 删除元素的个数[, 新元素1, 新元素2...]]) 返回: 被删除元素组成的新数组

let a = [1, 2];
a.splice(2, 0, 3);
a; // [1, 2, 3]
a.splice(1, 1);
a; // [1, 3]
a.splice(1, 1, 2); 
a; // [1, 2]
# Array.prototype.sort()

作用: 按照一定方法排序数组 参数: ([比较函数]) 返回: 处理后的数组

[3, 2, 5, 8, 4].sort(); // [2, 3, 4, 5, 8]
[3, 2, 5, 8, 4].sort((a, b)=> b - a); // [8, 5, 4, 3, 2]
# Array.prototype.reverse()

作用: 反转数组元素 参数: 无 返回: 处理后的数组

# Array.prototype.fill()

作用: 填充修改数组 参数: (要填充的元素, 起始索引, 结束索引) 返回: 处理后的数组

# 3.2.2 增删改-原数组无变化

# Array.prototype.toString()

作用: 将数组转化为字符串 参数: 无 返回: 一个字符串

[1,2,3].toString(); // '1,2,3'
# Array.prototype.join()

作用: 将数组元素用指定分隔符连接成一个字符串 参数: ([分隔符]) 返回: 一个字符串

[1,2,3].join(); // '1,2,3'
[1,2,3].join('-') // '1-2-3'
# Array.prototype.concat()

作用: 将元素与数组自身拼接成一个新数组 参数: ([元素 1, 元素 2...]) 返回: 处理后的新数组

[1,2,3].concat(); // [1,2,3]
[1,2,3].concat([4], [5,6]); // [1,2,3,4,5,6]
[1,2,3].concat(4); // [1,2,3,4]
[].concat({}); // [{}]
# Array.prototype.slice()

作用: 截取数组中的一段 参数: (起始索引[, 结束索引]) 返回: 截取的新数组

[1,2,3].slice() // [1,2,3]
[1,2,3].slice(1) // [2,3]
[1,2,3].slice(1,2) // [2]
# Array.prototype.flat()

作用: 按照一定深度扁平化数组 参数: ([深度值]) 返回: 扁平化后的新数组

[1,[2,[3,4]]].flat(); // [1,2,[3,4]]
[1,[2,[3,4]]].flat(Infinity); // [1,2,3,4]

# Array.prototype.includes()

# 3.2.4 数组遍历

# Array.prototype.forEach()

作用: 遍历数组，对每一个元素执行回调函数 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: undefined

# Array.prototype.map()

作用: 遍历数组，按照一定方法加工数组 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 加工后的新数组

[1,2,3].map(item => item * 2); // [2,4,6]
# Array.prototype.filter()

作用: 遍历数组，按照一定方法过滤数组 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 过滤后的新数组

[1,2,3].filter(item => item > 1); // [2,3]
# Array.prototype.reduce()

作用: 遍历数组，按照一定方法汇总元素 参数: (回调函数(汇总的值, 当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 汇总的值

[1,2,3].reduce((sum, item) => console.log(item)) // 2 3
# Array.prototype.find()

作用: 遍历数组，按照一定方法寻找匹配的第一个元素 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 一个元素, 或者 undefined

[1,2,3].find(item => item > 1); // 2
# Array.prototype.findIndex()

作用: 遍历数组，按照一定方法寻找匹配的第一个元素的索引 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 一个元素的索引, 或者 -1

[1,2,3].findIndex(item => item > 1); // 1
# Array.prototype.some()

作用: 遍历数组，按照一定方法测试元素, 并判断是否至少有一个通过 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 一个布尔值

[1,2,3].some(item => item > 1); // true
# Array.prototype.every()

作用: 遍历数组，按照一定方法测试元素, 并判断是否全部通过 参数: (回调函数(当前元素, 当前索引, 当前数组), 回调函数 this 指向) 返回: 一个布尔值

[1,2,3].every(item => item > 1); // false
