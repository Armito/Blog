# video

# 一. <video> 标签

# 1.1 标签用法

<video> 标签是一个双标签, 通过 src 属性引入视频

<video src="./xxx/xxx.mp4">
    你的浏览器不支持 video 标签
</video>

<video> 标签中可以使用 <source> 标签引入不同格式的相同视频

<video>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    您的浏览器不支持 video 标签
</video>

# 1.2 标签属性

属性 值 作用
src url 引入视频
controls controls 显示视频控件
poster url 引入海报
preload preload 预加载
autoplay autoplay 自动播放
loop loop 循环播放
muted muted 静音
width pixels 视频宽度
height pixels 视频高度

# 二. video 对象

# 2.1 video 对象的属性

属性 作用
src 设置或返回视频的 src 属性的值
controls 设置或返回视频是否应该显示控件
poster 设置或返回视频的 poster 属性的值
preload 设置或返回视频的 preload 属性的值
readyState 返回视频当前的就绪状态
autoplay 设置或返回是否在就绪后随即播放视频
loop 设置或返回视频是否应在结束时再次播放
duration 返回视频的时长(s)
currentTime 设置或返回视频当前播放时长(s)
playbackRate 设置或返回视频播放的速度
paused 设置或返回视频是否暂停
muted 设置或返回是否关闭声音
volume 设置或返回视频的音量
ended 返回视频的播放是否已结束

# 2.2 video 对象的方法

方法 作用
play() 开始播放视频
pause() 暂停当前播放的视频
load() 重新加载视频元素

# 2.3 全屏 API

  • 全屏属性和方法
属性/方法 作用
document.fullscreenElement 返回文档中处于全屏的元素, 或者 null
document.fullscreenEnabled 返回文档是否支持全屏
element.requestFullscreen() 进入全屏模式
document.exitFullscreen() 推出全屏模式
  • 全屏事件
// 必须绑定在 document 上, 发生在当进入全屏或退出全屏时
document.addEventListener('fullscreenchange', function () {

});
// onwebkitfullscreenchange, onmozfullscreenchange

# 三. 媒体事件

事件 触发时机
onerror 当在文件加载期间发生错误时
oncanplay 当文件就绪可以开始播放时
onplay 当媒介已就绪可以开始播放时
onplaying 当媒介已开始播放时
onpause 当媒介被用户或程序暂停时
ontimeupdate 当播放位置改变时
ondurationchange 当媒介时长改变时
onvolumechange 当音量改变时
onended 当媒介已到达结尾时
