JSX语法 推荐 / 支持JSX，推荐模板语法
创建组件 函数式组件/类组件(必须返回JSX) 同page一样生成component Vue.extend()/Vue.component()
使用组件 函数名/类名 component配置component，page配置usingComponents 实例/注册名

条件渲染 原生JS wx:if/wx:else-if/we:else v-if/v-else-if/v-else
列表渲染 原生JS wx:for v-for
更新渲染 setState() setData() 数据变化自动更新
事件绑定 驼峰命名法/采用事件代理 bind v-on,@/对于原生DOM元素和.native修饰符，采用addEventListener();对于其他组件，采用$on发布订阅模式
组件通讯/父->子 props properties props
组件通讯/子->父 自定义事件，函数传值 自定义事件，发布订阅/triggerEvent() 自定义事件，发布订阅/$emit()
组件通讯/兄弟 状态提升 / 事件中心bus
组件通讯/复杂 Redux getApp() Vuex
组件通讯/父->子深层 Context/Provider, Consumer组件 / 依赖注入/provide, inject
props校验 导入prop-types/组件名.propTypes = {PropTypes.xxx(string/isRequired/shape()等)}/组件名.defaultProps = {} xxx: {type, optinalTypes, value, observer} xxx: {type, isRequired, default等}
